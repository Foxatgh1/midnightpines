# Midnight Pines

A playable browser game for the "Vibe Coding a Game" assignment.

## Run locally

1. Open `/Users/colemanmartinis/Documents/New project/index.html` in a browser.
2. Click **Start / Restart**.
3. Move with **WASD** or **Arrow keys**.
4. Collect all **4 relics**.
5. After collecting all relics, touch creatures to eliminate them and win.

## Game concept and unique features

You are a yellow character running through haunted woods while being hunted by 4 creatures with different AI patterns:

- **Wanderer**: random movement, fastest creature.
- **Blinker**: always chases, slowest creature, teleports near you when far away.
- **Stalker**: direct chaser, second slowest creature.
- **Shifter**: alternates between direct chase and random movement.

Win condition: collect 4 relics, then eliminate all 4 creatures by touching them.
Loss condition: being touched before all relics are collected.

## Publish to get a public link

## Option 1: Netlify Drop (fastest)

1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop).
2. Drag the `index.html` file onto the page.
3. Netlify gives you a public URL to submit.

## Option 2: GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`.
3. In repo settings, enable **Pages** from the `main` branch root.
4. Use the generated GitHub Pages URL as your public link.
